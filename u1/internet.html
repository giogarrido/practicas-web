Conceptos básicos de internet

Preguntas

  1. ¿Qué es internet?

  2. ¿Qué son los proveedores de internet?

  3. Tipos de proveedores de internet?

  4. ¿Qué es la experiencia de usuario?

  5. ¿Qué es la interfaz de usuario?

  6. ¿Qué son los protocoos de internet?

  7. ¿Qué es HTML?

  8. ¿Qué es la semantica HTML?


Respuesta 1

Internet es una red mundial formada por millones de computadoras de todo tipo y plataforma, conectadas entre sí por diversos medios y equipos de comunicación que mediante una serie de protocolos (TCP/IP) hacen posible que los usuarios podamos localizar, seleccionar e intercambiar información.



Respuesta 2

Son empresas dedicadas a conectar a Internet a los usuarios, o las distintas redes que tengan, y a dar el mantenimiento necesario para que el acceso funcione correctamente. También ofrece servicios relacionados, como alojamiento web o registro de dominios, entre otros.

Respuesta 4

Se refieren a un conjunto de factores relativos a la interacción del usuario con un entorno o un dispositivo en concreto cuyo resultado es la generación de una percepción positiva o negativa de dicho servicio.



Respuesta 5

La interfaz de usuario es el medio con que el usuario puede comunicarse con una máquina, equipo, computadora o dispositivo, y comprende todos los puntos de contacto entre el usuario y el equipo. (Botones, imágenes, enlaces, menu etc.)



Respuesta 6

Conjunto de reglas que permiten que dos o mas entidades de un sistema se comuniquen entre ellos para trasmitir información.



Respuesta 7

(Lenguaje de Marcas de Hipertexto, del inglés HyperText Markup Language) es el componente más básico de la Web. Define el significado y la estructura del contenido web. Además de HTML, generalmente se utilizan otras tecnologías para describir la apariencia/presentación de una página web (CSS) o la funcionalidad/comportamiento (JavaScript).



Respuesta 8

El HTML semántico es un concepto para estructurar y escribir código HTML, de manera que el significado del contenido de una página web sea más claro a través de sus etiquetas.
